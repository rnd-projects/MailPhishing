# Mail Phishing App

A WPF project created to send a phishing email to CCC Distribution lists.

#### TECHNOLOGIES

  * C# 4.5
  * Entity Framework 6.1.1
  * Extended WPF Toolkit 2.0.0
  * WPF Rich Text 1.0.16.2
